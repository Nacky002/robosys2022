# robosys2022

## コマンド
![test](https://github.com/Nacky002/robosys2022/actions/workflows/test.yml/badge.svg)

* plus
  * 標準入力から読み込んだ数字の、総和を出力する。

* avg
  * 標準入力から読み込んだ数字の、平均値を出力する。

## 必要なソフトウェア
* Python
  * テスト済のバージョン : 3.7 ~ 3.10

## テスト環境
* Ubuntu 20.04.5 LTS
